# Memobase — memodb

**项目：** Memobase (memodb)
**来源：** arXiv 2512.13564v2 (Memory in the Age of AI Agents)
**类别：** Memory Framework — Structured Profile Memory

---

## 概述

以结构化profiles为核心的记忆管理框架。Memobase将用户记忆组织为结构化的profile，而非松散的对话历史或知识图谱节点。

## 核心机制

- **结构化Profiles** — 将用户信息、偏好、历史行为等组织为结构化的profile数据，作为记忆管理的核心单元
- **Profile驱动的记忆检索** — 基于结构化profile进行精确的记忆匹配和检索
- **记忆整合与更新** — 持续从对话中提取信息更新用户profile
