## 问题

多Agent协作中的记忆管理、时序一致性与长程推理。

## 方法

参考大脑认知机制设计多智能体记忆框架，模拟海马体-新皮层的双记忆系统，结合前额叶皮层的任务协调逻辑。模块化设计：
- 海马体启发的情景记忆模块
- 新皮层启发的语义记忆模块
- 协调模块（记忆整合与检索）

## 独特概念

提出"灵魂侵蚀（soul erosion）"诊断视角，定义为时间连贯性、语义一致性和身份保全三维度的综合退化。针对每种模式设计专用方案：
- 时间侵蚀 → StoryArc时间线索引
- 语义侵蚀 → 海马体到颞叶的整合机制
- 身份侵蚀 → 杏仁核启发的显著性标记

## 结果

- LoCoMo基准准确率78.45%
- 跨会话任务集成准确率52.6%
- 消融：海马体式情景记忆是核心驱动因素（移除后-24.62%），其他子系统针对特定失效模式提供补充支持
