# Inside Out (PersonaTree) — 人民大学、MemTensor、上海高级算法研究院

**论文：** [arXiv 2601.05171](https://arxiv.org/pdf/2601.05171)
**来源：** 机器之心Pro Week 07
**类别：** Agent Memory — 长期个性化对话

---

## 问题

长期个性化对话中记忆噪声积累、人格不一致、无限对话与有限上下文的冲突。

## 方法

借鉴认知科学中记忆的功能分工，将记忆分解为四大功能子系统：
- 情景记忆
- 语义记忆
- 显著性感知
- 控制导向

核心组件PersonaTree：基于生物心理社会模型构建分层记忆schema，设计迭代式树更新机制。训练轻量级MemListener模型，通过过程奖励强化学习将对话流转化为ADD/UPDATE/DELETE树操作。

## 推理阶段

自适应生成pipeline：
- 快速模式 — 直接用PersonaTree保障低延迟
- 代理模式 — 在树结构约束下按需引入细节，平衡效率与个性化深度

## 结果

- PersonaTree在抑制上下文噪声、保持人格一致性上优于全文连接与现有个性化记忆系统
- 小模型MemListener可达大模型相当的记忆操作性能
- "小模型维护记忆、大模型负责生成"的协同范式
