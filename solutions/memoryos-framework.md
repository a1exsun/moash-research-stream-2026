# MemoryOS — 百家/BAI-LAB

**项目：** MemoryOS (BAI-LAB)
**来源：** arXiv 2512.13564v2 (Memory in the Age of AI Agents)
**类别：** Memory Framework — Hierarchical Memory Architecture

---

## 概述

层次化短期/中期/长期记忆架构。MemoryOS是百家（BAI-LAB）开发的开源记忆管理框架，采用类人的多层记忆分级设计。

## 核心机制

- **短期记忆（Short-term Memory）** — 保存当前会话的即时交互信息
- **中期记忆（Mid-term Memory）** — 存储跨会话但近期的重要信息，作为短期与长期之间的缓冲
- **长期记忆（Long-term Memory）** — 持久化存储经过整合的核心知识和用户偏好
- **层次化记忆管理** — 信息在三层记忆之间自动流转和整合
