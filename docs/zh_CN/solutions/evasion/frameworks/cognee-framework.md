# Cognee

**项目：** Cognee
**来源：** arXiv 2512.13564v2 (Memory in the Age of AI Agents)
**类别：** Memory Framework — Knowledge Graph-based Memory

---

## 概述

基于Knowledge Graph的记忆管理框架，支持多模态信息的记忆存储与检索。Cognee将记忆组织为知识图谱结构，利用图的语义关联能力增强记忆的组织和访问。

## 核心机制

- **Knowledge Graph记忆存储** — 将记忆信息组织为知识图谱，利用实体-关系-实体的三元组结构表示语义关联
- **多模态支持** — 支持文本、图像等多种模态的信息存储和检索，统一纳入知识图谱框架
- **图结构记忆检索** — 基于图的拓扑结构和语义关系进行记忆检索，支持多跳推理和关联发现
